#  Daily English Sentence Automation (n8n + Slack)
This project is on automatation workflow built with n8n that sends English sentences every morning and evening to Slack.

It helps user practice English consistently with minimal effort.

Every morning, the system automatically:

1. Generates English sentences
2. Sends it to a designated Slack channel or user
3. Allows daily exposure to natural English without manual input.
